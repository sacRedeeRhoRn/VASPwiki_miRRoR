# CONTCAR_disp-N
The CONTCAR_disp-N file is an output file of a [split finite differences
calculation](../tutorials/Restarting_finite_differences_calculations.md)
that contains the structural information of the *N*th displacement. It
has the same format as the [CONTCAR](CONTCAR.md) or
[POSCAR](../input-files/POSCAR.md) file.

|                                      |
|--------------------------------------|
| **Mind:** Available as of VASP 6.6.0 |
