# Category:Constrained-random-phase approximation
The **constrained random-phase approximation** (cRPA) is a method that
allows the calculation of the effective interaction parameter
$U$, $J$, and $J'$ for model
Hamiltonians. The main idea is to neglect the screening effects of
specific target states in the screened Coulomb interaction
$W$ of the [GW
method](../redirects/The_GW_approximation_of_Hedin's_equations.md).
The resulting partially screened Coulomb interaction is evaluated in a
[localized basis](https://vasp.at/wiki/index.php/Wannier_functions) that
spans the target space and is described by the model Hamiltonian. The
target space is usually low-dimensional and therefore allows for the
application of a higher-level theory, such as dynamical-mean-field
theory (DMFT).
