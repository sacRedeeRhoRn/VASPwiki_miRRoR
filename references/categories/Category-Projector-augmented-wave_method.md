# Category:Projector-augmented-wave method
The **projector-augmented-wave (PAW) method** is a specific kind of
[pseudopotential approach](../redirects/Pseudopotentials.md).
The [PAW formalism](../redirects/PAW_formalism.md) is the basis of
how VASP computes the Kohn-Sham orbitals and other quantities. An
important aspect of the PAW method is so-called augmentation that
regulates which part of the calculation is treated on the plane-wave
grid and which part on the radial logarithmic grid in the vicinity of
the nuclei.

This category contains pages specifically about the PAW method and
control tags for the PAW method that can be set in the
[INCAR](../input-files/INCAR.md) file, e.g.,
[LMAXMIX](../incar-tags/LMAXMIX.md).
