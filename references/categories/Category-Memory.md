# Category:Memory


**Memory** can be a limiting factor for VASP calculations of large
systems. However, there are some tags that can significantly reduce the
**memory** requirements and still achieve the same results. The
reduction often comes at the cost of an extended computational time,
though. Complimentary to **memory**, consider the
[parallelization](Category-Parallelization.md)
of the calculation. These topics strongly influence each other.
