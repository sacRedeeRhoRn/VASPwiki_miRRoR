# vaspin.h5


This file can be used to replace the typical
[INCAR](INCAR.md), [POSCAR](POSCAR.md),
[POTCAR](POTCAR.md), and [KPOINTS](KPOINTS.md)
files using a single HDF5 file.

Currently, we do not provide a tool to systematically create this file
from scratch, instead, this feature is currently mostly intended to
reproduce previous calculations.

A [vaspout.h5](../output-files/Vaspout.h5.md) file that resulted from
previous calculations can be renamed to
vaspin.h5, copied to an empty
directory, and used as input for a new VASP run.
