# POSCAR.prim


The POSCAR.prim file contains
the structural information of the primitive cell required for the <a
href="/wiki/index.php?title=K-point_projection_scheme&amp;action=edit&amp;redlink=1"
class="new"
title="K-point projection scheme (page does not exist)">k-point
projection scheme</a>.

The format of this file is similar to the
[POSCAR](../input-files/POSCAR.md) file.
