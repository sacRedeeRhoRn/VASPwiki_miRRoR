# POSCAR.prim
The POSCAR.prim file contains the structural information of the
primitive cell required for the [k-point projection
scheme](https://vasp.at/wiki/index.php/index.php)").

The format of this file is similar to the
[POSCAR](../input-files/POSCAR.md) file.
