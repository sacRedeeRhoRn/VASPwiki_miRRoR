# MINROT
MINROT = \[real\]  
Default: **MINROT** = 0.01 

Description: The flag MINROT defines the value for which the dimer is
rotated only if the predicted rotation angle is greater than MINROT
(rad.) in the [Improved Dimer
Method](../misc/Improved_Dimer_Method.md).
