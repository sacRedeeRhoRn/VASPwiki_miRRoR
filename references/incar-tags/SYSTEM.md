# SYSTEM
SYSTEM = \[string\]  
Default: **SYSTEM** = unknown system 

Description: The "title string" defined by SYSTEM is for the user only
and should help the user to identify what he/she wants to do with this
specific input file.

------------------------------------------------------------------------

The SYSTEM tag is followed by a string that possibly contains blanks.
The string is read in and written to the main output file
[OUTCAR](../output-files/OUTCAR.md).
