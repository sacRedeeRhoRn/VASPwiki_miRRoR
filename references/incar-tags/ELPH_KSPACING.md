# ELPH KSPACING


ELPH_KSPACING = \[real\]  
Default: **ELPH_KSPACING** = 0.5 

Description: The tag
ELPH_KSPACING determines the
**k** point mesh if the
[KPOINTS_ELPH](../input-files/KPOINTS_ELPH.md) file is not present.

|  |
|----|
| **Mind:** Available as of VASP 6.5.0 |

------------------------------------------------------------------------

For additional details of how the **k** point mesh is generated, see the
documentation of the [KSPACING](KSPACING.md) tag.
