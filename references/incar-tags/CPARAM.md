# CPARAM


CPARAM = \[real\]  
Default: **CPARAM** = 0.0093 

Description: The tag CPARAM
allows to specify the value of the parameter $C$ in the
kernel of the nonlocal rVV10 correlation
functional.<sup>[\[1\]](#cite_note-sabatini:prb:2013-1)</sup>
