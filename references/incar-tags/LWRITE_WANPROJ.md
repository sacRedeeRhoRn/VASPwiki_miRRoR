# LWRITE_WANPROJ


LWRITE_WANPROJ = \[logical\]  
Default: **LWRITE_WANPROJ** = .FALSE. 

Description: LWRITE_WANPROJ
determines whether the Wannier projection file
[WANPROJ](../input-files/WANPROJ.md) is written.

------------------------------------------------------------------------

If [WANPROJ](../input-files/WANPROJ.md) is present, VASP skips the Wannier
projection and reads the effective projectors U<sub>nk</sub> from the
[WANPROJ](../input-files/WANPROJ.md) file for each k point.
