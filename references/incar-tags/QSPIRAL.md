# QSPIRAL


QSPIRAL = \[real array\]  
Default: **QSPIRAL** = 3\*0.0 

Description: the QSPIRAL-tag
specifies the spin spiral propagation vector.

------------------------------------------------------------------------

QSPIRAL specifies the spin
spiral propagation vector in direct coordinates of the reciprocal
lattice.

See the [description of spin spiral
calculations](../theory/Spin_spirals.md) for more details.
