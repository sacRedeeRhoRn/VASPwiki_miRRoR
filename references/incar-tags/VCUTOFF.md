# VCUTOFF


VCUTOFF = \[real\]  
Default: **VCUTOFF** = 1.1 [ENCUTGW](ENCUTGW.md) 

  
Description: The parameter
VCUTOFF sets the energy cutoff
for bare Coulomb matrix elements and controls the basis set for the bare
Coulomb interaction.
