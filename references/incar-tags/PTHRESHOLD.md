# PTHRESHOLD
PTHRESHOLD = \[real\] 

Description: Control flag for the output of the profiling routines.

------------------------------------------------------------------------

If specified all routines with less execution time than
PTHRESHOLD=*time* are not printed out in the
[OUTCAR](../output-files/OUTCAR.md) file.
