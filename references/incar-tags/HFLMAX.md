# HFLMAX
HFLMAX = \[integer\]  
Default: **HFLMAX** = 4 

Description: To be compatible w.r.t. old releases, VASP also reads the
flag HFLMAX to the same effect as [LMAXFOCK](LMAXFOCK.md).
