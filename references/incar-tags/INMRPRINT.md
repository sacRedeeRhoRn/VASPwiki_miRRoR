# INMRPRINT
INMRPRINT  = -1 \| 0 \| 1  
Default: **INMRPRINT** = 0 

Description: Determines the verbosity of the
[NMR](../categories/Category-NMR.md) output.

|                                       |
|---------------------------------------|
| **Mind:** Available as of VASP.6.6.0. |

------------------------------------------------------------------------

The NMR output is written to the [OUTCAR](../output-files/OUTCAR.md).
Setting INMRPRINT 0 is the default, while 1 prints more, and -1 prints
less.
