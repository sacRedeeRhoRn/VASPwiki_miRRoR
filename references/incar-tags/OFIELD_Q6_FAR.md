# OFIELD_Q6_FAR
OFIELD_Q6_FAR = \[real\] 

Description: OFIELD_Q6_FAR sets the far fading distance for the
computation of a continuous $Q_6$
parameter in the
[interface-pinning](../categories/Category-Interface_pinning.md)
method.

------------------------------------------------------------------------

The unit of OFIELD_Q6_FAR is $\AA$.
