# NRMM


NRMM = \[integer\]  
Default: **NRMM** = 4 

Description: Maximum number of iterative steps for RMM-DIIS
([IALGO](IALGO.md)=4X) or Davidson algorithms
([IALGO](IALGO.md)=3X).

------------------------------------------------------------------------

Note: This tag was formerly known as NDAV which is the name of the
variable in the VASP source code.
