# CH_AMPLIFICATION
CH_AMPLIFICATION = \[real\]  
Default: **CH_AMPLIFICATION** = 1.0 

Description: This tag is used to scale (amplify) the calculated
dielectric function within the supercell core-hole method and the BSE+GW
method for core electrons.
