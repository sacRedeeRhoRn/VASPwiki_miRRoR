# VDW_R0AU
VDW_R0AU = \[real (array)\] 

Description: VDW_R0AU defines the $R_0$
parameters (bohr) for each species defined in the
[POSCAR](../input-files/POSCAR.md) file.

------------------------------------------------------------------------

VDW_R0AU sets the $R_0$ parameters
(bohr) for each species defined in the [POSCAR](../input-files/POSCAR.md)
file for the
[Tkatchenko-Scheffler](../methods/Tkatchenko-Scheffler_method.md)
and [Many-body dispersion
energy](../methods/Many-body_dispersion_energy.md)
methods.
