# LWAVE


LWAVE = \[logical\]  
Default: **LWAVE** = .TRUE. 

Description: Determines whether the wavefunctions are written at the end
of a run.

------------------------------------------------------------------------

The wavefunctions are written to the [WAVECAR](../input-files/WAVECAR.md)
file by default. If [`LH5`](LH5.md)` = T`, the wavefunctions
are written to the [vaspwave.h5](../output-files/Vaspwave.h5.md)
instead.
