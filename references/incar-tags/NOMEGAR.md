# NOMEGAR


NOMEGAR = \[integer\] 

|  |  |  |
|----|----|----|
| Default: **NOMEGAR** | = [NOMEGA](NOMEGA.md) | for <a href="/wiki/GW_calculations" class="mw-redirect"
title="GW calculations">GW calculations</a> |
|  | = 0 | for <a href="/wiki/ACFDT_calculations" class="mw-redirect"
title="ACFDT calculations">ACFDT calculations</a> |

Description: NOMEGAR specifies
the number of frequency grid points along the real axis.

------------------------------------------------------------------------

Usually NOMEGAR equals
[NOMEGA](NOMEGA.md). If
NOMEGAR is smaller than
[NOMEGA](NOMEGA.md) (for instance 0), frequencies along the
imaginary time axis are included (this feature is currently not fully
supported).
