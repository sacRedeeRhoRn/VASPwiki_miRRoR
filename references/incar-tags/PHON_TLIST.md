# PHON_TLIST
PHON_TLIST = \[real array\]  
Default: **PHON_TLIST** = none 

Description: Provides the list of temperatures for that the
electron-phonon interaction using the ZG
configuration^([\[1\]](#cite_note-zacharias:prb:2016-1)) is evaluated.

------------------------------------------------------------------------

For further usage of this tag see: [Electron-phonon interactions from
Monte-Carlo
sampling](../tutorials/Electron-phonon_interactions_from_Monte-Carlo_sampling.md).

|                                                       |
|-------------------------------------------------------|
| **Mind:** This feature is available for VASP \>= 6.0. |
