# OFIELD_A


OFIELD_A = \[real\] 

Description: The tag OFIELD_A
sets the desired order parameter $Q_6$ in the
[Interface
pinning](../categories/Category-Interface_pinning.md)
method.

------------------------------------------------------------------------

The bias potential in the [Interface
pinning](../categories/Category-Interface_pinning.md)
method is written as

$U_\text{bias}(\mathbf{R}) = \frac\kappa2 \left(Q_6(\mathbf{R}) -
A\right)^2$.

The tag OFIELD_A sets the
desired value of the order parameter $A$.
