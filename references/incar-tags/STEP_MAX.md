# STEP_MAX
  
STEP_MAX = \[real\]  
Default: **STEP_MAX** = 0.1 

Description: The flag STEP_MAX defines the trust radius (upper limit)
for the optimization step (in $\AA$ ) in
the [Improved Dimer
Method](../misc/Improved_Dimer_Method.md).
