# STEP_MAX


  
STEP_MAX = \[real\]  
Default: **STEP_MAX** = 0.1 

Description: The flag STEP_MAX
defines the trust radius (upper limit) for the optimization step (in
$\AA$ ) in the
<a href="/wiki/Improved_Dimer_Method" class="mw-redirect"
title="Improved Dimer Method">Improved Dimer Method</a>.
