# OFIELD_Q6_NEAR


OFIELD_Q6_NEAR = \[real\] 

Description: OFIELD_Q6_NEAR
sets the near-fading distance (in $\AA$) for the
computation of a continuous $Q_6$
parameter in the
[interface-pinning](../categories/Category-Interface_pinning.md)
method.
