# CH_NEDOS


CH_NEDOS = \[integer\]  
Default: **CH_NEDOS** = 1000 

Description: This tag specifies the number of frequency (energy) grid
points on the x-axis in the calculation of the dielectric function in
the supercell core-hole method.
