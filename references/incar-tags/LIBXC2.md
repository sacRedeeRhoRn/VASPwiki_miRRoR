# LIBXC2


LIBXC2 = \[string\] or
\[integer\] 

Description: LIBXC2 specifies
the correlation functional from the library of exchange-correlation
functionals
Libxc[^marques:cpc:2012-1][^lehtola:sx:2018-2][^tran:arxiv:2026-3][^libxc-4].
LIBXC2 can be used only if the
functional specified with [LIBXC1](LIBXC1.md) corresponds to
only exchange and not to exchange and correlation.

------------------------------------------------------------------------

A detailed explanation of how to use [LIBXC1](LIBXC1.md) and
LIBXC2 and examples can be
found on the [LIBXC1](LIBXC1.md) page.
