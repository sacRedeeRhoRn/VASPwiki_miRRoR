# PHON_LMC


PHON_LMC = \[logical\]  
Default: **PHON_LMC** = .FALSE. 

Description: Controls whether electron-phonon interactions from
Monte-Carlo sampling are calculated or not.

------------------------------------------------------------------------

For further usage of this tag see: [Electron-phonon interactions from
Monte-Carlo
sampling](../tutorials/Electron-phonon_interactions_from_Monte-Carlo_sampling.md).
