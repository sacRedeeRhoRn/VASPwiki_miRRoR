# PFLAT
[PMASS](PMASS.md) = \[logical\] 

Description: Control flag for the output of the profiling routines.

------------------------------------------------------------------------

If specified as [PMASS](PMASS.md)=*.TRUE.*, only the flat
profile is printed out in the [OUTCAR](../output-files/OUTCAR.md).
