# LELF


LELF = .TRUE. \| .FALSE.  
Default: **LELF** = .FALSE. 

Description: LELF determines
whether to create an [ELFCAR](../output-files/ELFCAR.md) file or not.

------------------------------------------------------------------------

This file contains the so-called ELF (electron localization function)
<sup>[\[1\]](#cite_note-silvi:nature:371-1)</sup>

If LELF is set,
[NPAR](NPAR.md)=1 has to be set explicitely in the
[INCAR](../input-files/INCAR.md) file in addition
