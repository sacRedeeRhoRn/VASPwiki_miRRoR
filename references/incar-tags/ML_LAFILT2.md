# ML_LAFILT2


ML_LAFILT2 = \[logical\]  
Default: **ML_LAFILT2** = .TRUE. 

Description: This tag specifies whether angular filtering is applied or
not within the machine learning force field method.

------------------------------------------------------------------------

For the desciption of the filters please see:
[ML_IAFILT2](ML_IAFILT2.md) and
[ML_AFILT2](ML_AFILT2.md).

The applied angular filter is also briefly described
[here](../methods/Machine_learning_force_field-_Theory.md).
