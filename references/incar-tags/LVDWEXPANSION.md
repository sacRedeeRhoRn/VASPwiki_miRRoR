# LVDWEXPANSION


LVDWEXPANSION = .TRUE. \|
.FALSE.  
Default: **LVDWEXPANSION** = .FALSE. 

Description: LVDWEXPANSION 
decides whether to write the two- to six- body contributions to MBD
dispersion energy in the [OUTCAR](../output-files/OUTCAR.md) file.
