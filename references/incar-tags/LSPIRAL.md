# LSPIRAL
LSPIRAL = .TRUE. \| .FALSE.  
Default: **LSPIRAL** = .FALSE. 

Description: set LSPIRAL=.TRUE. to represent spin spirals by means of a
generalized Bloch condition.

------------------------------------------------------------------------

See the [description of spin spiral
calculations](../theory/Spin_spirals.md) for more details.
