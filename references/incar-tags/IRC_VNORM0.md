# IRC_VNORM0


IRC_VNORM0 = real  
Default: **IRC_VNORM0** = 0.002 

Description: The constant velocity vector (in Å/fs) in an
<a href="/wiki/IRC_calculations" class="mw-redirect"
title="IRC calculations">IRC calculation</a>.

------------------------------------------------------------------------

In the damped-velocity-Verlet algorithm for
<a href="/wiki/IRC_calculations" class="mw-redirect"
title="IRC calculations">IRC calculations</a>, the damping is realized
via rescaling the velocity vector to a constant value (in Å/fs) defined
via the parameter IRC_VNORM0.
