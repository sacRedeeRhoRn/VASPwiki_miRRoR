# LIBXC2_Pn


LIBXC2_Pn = \[real\] 

Description: LIBXC2_Pn, where
$n=1, 2, \ldots$ allows to specify the values of the
parameters of the functional implemented in Libxc that is called with
[LIBXC2](LIBXC2.md).

------------------------------------------------------------------------

A detailed explanation on how to use
[LIBXC1_Pn](LIBXC1_Pn.md) and
LIBXC2_Pn and an example can
be found at the [LIBXC1_Pn](LIBXC1_Pn.md) page.
