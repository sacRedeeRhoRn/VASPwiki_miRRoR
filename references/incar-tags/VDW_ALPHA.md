# VDW_ALPHA


VDW_ALPHA = \[real (array)\] 

Description: VDW_ALPHA defines
the polarizability $\alpha_{0}$
(bohr$^{3}$) for
each species defined in the [POSCAR](../input-files/POSCAR.md) file.

------------------------------------------------------------------------

VDW_ALPHA sets the
polarizability $\alpha_{0}$
(bohr$^{3}$) for
each species defined in the [POSCAR](../input-files/POSCAR.md) file for the
[Tkatchenko-Scheffler](../methods/Tkatchenko-Scheffler_method.md)
and [Many-body dispersion
energy](../methods/Many-body_dispersion_energy.md)
methods.
