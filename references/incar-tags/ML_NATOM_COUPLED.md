# ML_NATOM_COUPLED


ML_NATOM_COUPLED =
\[integer\]  
Default: **ML_NATOM_COUPLED** = 0 

Description: This tag specifies the number of atoms for which a coupling
parameter is introduced to calculate the chemical potential within the
machine learning force field method.

------------------------------------------------------------------------

For thermodynamic integration calculations please have a look at the tag
[ML_LCOUPLE](ML_LCOUPLE.md).
