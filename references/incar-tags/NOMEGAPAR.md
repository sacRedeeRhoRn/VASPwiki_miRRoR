# NOMEGAPAR


NOMEGAPAR = \[integer\] 

|  |  |  |
|----|----|----|
| Default: **NOMEGAPAR** | = NOMEGA | used in low scaling [GW](../methods/GW_approximation_of_Hedin's_equations.md) and RPA/ACFDT calculations. |

Description: NOMEGAPAR
available as of VASP.6, specifies the number of processor groups sharing
the same imaginary frequency grid points.
