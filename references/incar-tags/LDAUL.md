# LDAUL


LDAUL = \[integer array\]  
Default: **LDAUL** = NTYP\*2 

Description: LDAUL specifies
the $l$-quantum
number for which the on-site interaction is added.

------------------------------------------------------------------------

For LDAUL=-1 no on-site
interaction is added.

**Mind**: one *has* to specify one number for each atomic species.
