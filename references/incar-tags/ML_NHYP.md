# ML_NHYP
ML_NHYP = \[integer\]  
Default: **ML_NHYP** = 4 

Description: This tag specifies the polynomial power
$\zeta$ of the kernel within the machine
learning force field method (see [this
section](../methods/Machine_learning_force_field-_Theory.md)).

------------------------------------------------------------------------

The value of ML_NHYP must be an integer number $\ge$ 1.
