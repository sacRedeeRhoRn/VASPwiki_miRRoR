# NBSEEIG
NBSEEIG = \[integer\]  
Default: **NBSEEIG** = 0 

Description: NBSEEIG sets the number of BSE eigenvectors written to the
[BSEFATBAND](../output-files/BSEFATBAND.md) file. If
[BSEHOLE](BSEHOLE.md) or
[BSEELECTRON](BSEELECTRON.md) is set, NBSEEIG exciton
wavefunctions are written to the [CHG](../output-files/CHG.md).XX files.
