# STEP_SIZE
  
STEP_SIZE = \[real\]  
Default: **STEP_SIZE** = 0.01 

Description: The flag STEP_SIZE defines the trial step size for the
optimization step (in $\AA$ ) in the
[Improved Dimer
Method](../misc/Improved_Dimer_Method.md).
