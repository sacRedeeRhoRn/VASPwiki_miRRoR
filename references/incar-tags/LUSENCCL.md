# LUSENCCL


LUSENCCL = \[logical\]  
Default: **LUSENCCL** = .TRUE. 

Description: LUSENCCL allows
deactivating the NVIDIA Collective Communications Library (NCCL) during
runtime when the code was compiled with the
[-DUSENCCL](../misc/Precompiler_options.md)
option.
