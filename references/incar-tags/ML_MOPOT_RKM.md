# ML_MOPOT_RKM


ML_MOPOT_RKM = \[real\]  
Default: **ML_MOPOT_RKM** = 1.0 

Description: Specifies the width parameter of the Morse potentials used
as auxiliary potentials in thermodynamic integration
([VCAIMAGES](VCAIMAGES.md)).

------------------------------------------------------------------------

This parameter sets the width parameter $\beta$ of the
following Morse potential

$V(r) = D_e (1-e^{-\beta(r-r_{e})})^2 - 1.$
