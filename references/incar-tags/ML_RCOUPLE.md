# ML_RCOUPLE


ML_RCOUPLE = \[real\] 

|  |  |  |
|----|----|----|
| Default: **ML_RCOUPLE** | = 0.0 | if [ML_LEMPPOT](ML_LEMPPOT.md)=.TRUE. |
|  | = 1.0 | otherwise |

Description: This tag specifies the value of the coupling parameter for
the calculation of the chemical potential within the machine learning
force field method.

------------------------------------------------------------------------

For thermodynamic integration calculations please have a look at the tag
[ML_LCOUPLE](ML_LCOUPLE.md).
